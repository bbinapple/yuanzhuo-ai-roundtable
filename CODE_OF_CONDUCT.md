# Code of Conduct

## Our Pledge

We want Yuanzhuo AI Roundtable to be a welcoming, respectful, and useful open-source project for people building human-in-the-loop AI workflows.

Participants are expected to communicate with patience and good faith, especially when discussing safety, privacy, prompt design, deployment risk, and user-facing behavior.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but clarify risk directly.
- Keep issues and pull requests focused on the project.
- Do not post API keys, private meeting notes, customer data, or other sensitive material.
- When reporting security issues, use the private security process described in `SECURITY.md`.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Posting private credentials, personal information, or confidential business data.
- Repeated off-topic comments after maintainers ask to refocus.
- Encouraging unsafe deployment patterns without appropriate warnings.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, or pull requests that violate this code of conduct. Maintainers may also block users who repeatedly act in bad faith or create safety risks for the community.

## Scope

This code applies to project spaces such as GitHub issues, pull requests, discussions, documentation, and related community channels.
