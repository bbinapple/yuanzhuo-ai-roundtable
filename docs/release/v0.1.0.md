# v0.1.0 Release Notes

Yuanzhuo AI Roundtable v0.1.0 is the first public open-source package.

## Highlights

- Local-first FastAPI backend and browser UI.
- Bring-your-own-key OpenAI-compatible API settings.
- Multi-agent roundtable debate with analyst, executor, optional third debater, moderator, researcher, and secretary roles.
- Topic templates for free discussion, SWOT decisions, negotiation prep, side project review, and ecommerce opportunity review.
- Streaming responses, user interjections, pause/resume, summaries, todos, tags, scores, and Markdown export.
- Two UI entry points: single-panel workspace and dual-panel comparison workspace.
- MIT License, contributing guide, security policy, privacy notes, deployment notes, and validation script.

## Safety Notes

- No maintainer-owned provider key is included.
- `.env`, local histories, caches, logs, exports, and agent workspace metadata are excluded by `.gitignore`.
- `read_local` is disabled by default and is omitted from the model tool list unless `YUANZHUO_ENABLE_READ_LOCAL=1` is set.
- Tool calls require user confirmation by default through `YUANZHUO_TOOL_CONFIRM_THRESHOLD=0`.
- Public or shared deployments should add HTTPS, authentication, rate limits, and tool-access review.

## Known Limitations

- The UI is bilingual in key paths, but not a full i18n implementation yet.
- Docker packaging is not included in v0.1.0.
- LocalStorage is used for browser-side API settings; shared deployments need a stronger credential model.
- Example workflows are intentionally generic and do not promise financial, legal, medical, or commercial outcomes.
