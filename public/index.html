Quick Start / 首次使用说明

Get started with AI Roundtable / 开始使用圆桌会议

This version does not include a site-owner API key. Each user must configure their own API before starting a debate. / 本版本不内置站长 API Key，需配置自己的 API 后使用。

1
API Settings / 填写 API 设置

Enter Base URL and your own API key. The key is stored only in this browser. / 输入 Base URL 和自己的 API Key。

2
Default Models / 选择默认模型

Choose models for analyst, executor, and secretary roles. / 设置分析型、执行型和秘书使用的模型。

3
Agents / 选择 Agent

A/B are the default debaters. C is optional. / A/B 是默认两位辩手，C 是可选第三辩手。

4
Start / 输入议题开始

Enable preheat research or start directly, then export summaries and action items. / 可开启预热，也可以直接开辩。

Use HTTPS for any public deployment to protect API keys in transit. / 公网部署建议使用 HTTPS。
